# Results

This folder contains summarized experimental results obtained during the Bachelor Project.

The project evaluated three scenarios:

- Baseline Operation
- Denial of Service (DoS)
- False Data Injection (FDI)

The analysis focused on:

- Latency
- Jitter
- Packet Loss
- Inter-Packet Delay Variation (IPDV)

Detailed methodology, statistical analysis, and findings are documented in the final project report.
